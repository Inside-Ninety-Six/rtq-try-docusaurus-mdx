                    &= \dfrac {1} {4} \ \text{hours}
\end{aligned}

Let's find the speed now

$
\begin{aligned}
\text{Speed}    &= \dfrac {\text{Distance}} {\text{Time}} \\\\
                &= \dfrac {4} { \dfrac {1}{4} } \ \text{km/hour} \\\\
                &= 16 \ \text{km/hour}
\end{aligned}
$
