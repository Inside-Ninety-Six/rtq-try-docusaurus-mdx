                        If March $23 \text{rd}$ is a Monday, on what day of the week is April $2 \text{nd}$?

                        The month March has $31$ days.

                        A week has $7$ days.

                        $ \begin{aligned} & = \text{March } 23 \ \text{Monday} + 7 \ \text{days} + 1 \ \text{day} + 2 \ \text{days} \\ & = \text{March } 30 \ \text{Monday} + 1 \ \text{day} + 2 \ \text{days} \\ & = \text{March } 31 \ \text{Tuesday} + 2 \ \text{days} \\ & = \text{April } 2 \ \text{Thursday}
                        \end{aligned} $

                        $\text {April \ 2 \text{nd} falls on a = \ \text{Thursday}$

                        Thursday
