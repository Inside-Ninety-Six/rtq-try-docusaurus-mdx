In arithmetic questions, any calculations inside brackets are carried out first.

Put brackets in to make this calculation correct:

$3 + 5 \times 2 + 3 = 40$

</div>
<div class='workings'>
<div class='working'>

$
\begin{aligned}
&= 3 + 5 \times 2 + 3 \\\\
&= 3 + (5 \times 2) + 3 \\\\
&= 3 + (5 \times 2) + 3
\end{aligned}
$

</div>
</div>
<div class='answers'>
<div class='answer'>

$3 + (5 \times 2) + 3$

</div>
